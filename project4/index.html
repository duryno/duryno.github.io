Individual Part - Frederik

I have implemented the 'Display multiple hand-outlines and highlight the corresponding PCA-coordinates when one is selected' suggestion from the assignment description.

When the user clicks a point in the scatter plot, the hand corresponding to the point will be display in the hand-outline section. The user can click multiple points in the scatter plot, to display multiple hands. Each click generates a random color for the point and hand-outline pair, used to indicate which points has been clicked, and to illustrate which hand-outlines each point represents. The user can toggle the hand-outline, by clicking the point again.
